"Tableau dashboard analyzing global covid-19 with insights on cases, death, and recoveries".

# Project Overview
This project analyzes global COVID-19 data using Tableau.  
It shows total cases, deaths, recoveries, and active cases across different countries.

# Tools Used
- Tableau  
- Excel (Dataset)

# Key Insights
- Countries with highest recoveries  
- Comparison of total deaths vs cases  
- Global distribution using map visualization
